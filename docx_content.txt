一、项目背景
──────────────────────────────────────────────────────────────────────────────
- 项目: 厌氧型封膜机 下位机固件
- MCU:  STM32F405xx (ARM Cortex-M4, 168MHz)
- RTOS: FreeRTOS V10.3.1 + CMSIS-RTOS V2 封装
- 编译: MDK-ARM (Keil uVision 5)
- 主机通讯: Modbus RTU Slave, USART1, 115200bps, 从机地址默认 0x01
- 外设通讯: Modbus RTU Master, USART3 RS-485 → 4个从设备
  (吸盘0x01/吸膜电缸0x02/封膜电缸0x03/富俊电机0x04)
- 其他串口: USART2(PH板) USART4(温控) USART6(OD板)
- 固件版本: BetaV1.2.2

二、协议文档
──────────────────────────────────────────────────────────────────────────────
- 正式文档: 封膜机下位机通讯协议V1.0.docx (位于项目根目录)
- 提取版: docx_output.txt

四、代码架构（最终状态）
──────────────────────────────────────────────────────────────────────────────
三层架构，单向依赖:

  usart_comm.c (Core/Src/app/)           ← 协议传输层 (~260行)
    │ 纯 Modbus 组帧/解帧、CRC、功能码分发、FreeRTOS 任务
    │ 调用 ↓
  modbus_slave_reg.c (Core/Src/common/)  ← 寄存器业务层 (~360行)
    │ 寄存器映射表、读写分发、动作状态追踪(0x00A0)、故障位图(0x0002)
    │ 温度桩函数、复合动作预留桩(0x0060~0x0064)
    │ 调用 ↓
  app_control.c (Core/Src/app/)          ← 应用层 (~150行)
      485 外设初始化(appInitTask)、AT24C02 EEPROM 读写(eeprom_read/write_u32)

关键公开接口 (在 modbus_slave_reg.h 中声明):
  modbus_reg_lookup(addr)          — 二分查找寄存器定义
  modbus_reg_check_access(addr, a) — 检查寄存器可读/可写性
  modbus_reg_read_value(addr, &v)  — 读取寄存器值
  modbus_reg_write_execute(addr,v) — 执行16位写(0x06功能码)
  modbus_reg_write32_execute(a,v)  — 执行32位写(0x10功能码)
  modbus_get_slave_addr()          — 获取从机地址

六、Modbus 寄存器完整列表
──────────────────────────────────────────────────────────────────────────────
2.1 系统命令
  0x0000 RO  uint16  版本信息      → 101 = V1.01
  0x0001 RW  uint16  设备地址      → 1~247
  0x0002 RO  uint16  故障位图      → bit0=温度 bit1=吸膜电缸 bit2=封膜电缸 bit3=丝杆 bit4=吸盘

2.2 工作流程控制
  0x0010 WO  uint16  开始流程
  0x0011 WO  uint16  初始化流程
  0x0012 RO  uint16  系统状态      → [预留]

2.3 调试动作控制
  0x0020 RW  uint16  控制/查询温度  → [预留] 写=设温, 读=查温; 桩函数 temp_ctrl_set/get
  0x0030 WO  uint32  富俊电机移动   → 带到位检测(±500pulses,30s超时) + 写0x00A0+0x0002
  0x0040 WO  uint16  吸膜电缸移动   → 带到位检测(10s超时,堵转检测) + 写0x00A0+0x0002
  0x0041 WO  uint16  封膜电缸移动   → 带到位检测(10s超时,堵转检测) + 写0x00A0+0x0002
  0x0050 WO  uint16  电动吸盘开关   → 1=吸取(5s超时) 0=释放(5s超时) + 写0x00A0+0x0002
  0x0060 WO  uint16  富俊电机归零   → [预留] 桩函数 cmd_fujun_home, 写失败到0x00A0
  0x0061 WO  uint16  吸膜动作       → [预留] 桩函数 cmd_suction_action
  0x0062 WO  uint16  铺膜动作       → [预留] 桩函数 cmd_lay_film_action
  0x0063 WO  uint16  封膜动作       → [预留] 桩函数 cmd_seal_action
  0x0064 WO  uint16  取/放孔板      → [预留] 桩函数 cmd_well_plate_action

EEPROM 参数
  0x0070 RW  uint32  零点位置       → EEPROM_ADDR_ZERO    (0x00)
  0x0072 RW  uint32  吸膜/封膜位置  → EEPROM_SUCK_SEAL    (0x04)
  0x0074 RW  uint32  铺膜点位置     → EEPROM_PAVE         (0x08)
  0x0076 RW  uint32  取放孔板位置   → EEPROM_GET_PLACE    (0x0C)
  0x0078 RW  uint16  温度值         → EEPROM_TEMP_CTRL    (0x10)
  0x0079 RW  uint16  压膜时间       → EEPROM_PRESS_TIME   (0x14)

调试动作状态
  0x00A0 RO  uint16  执行状态       → 高字节=动作地址, 低字节: 0x00=成功 0x01=失败 0xFF=无历史
